# 说明

- 目录结构
```shell
-- argocd-manifests
  -- bj
    -- k8s-cluster-prod-001
        -- consul-agent
        -- consul-server
        -- ...
        -- ...
        -- ...
        -- ...
  -- gd
    -- k8s-cluster-prod-001
        -- ...
        -- ...
        -- ...
        -- ...
  -- hk
    -- k8s-cluster-prod-001
        -- ...
        -- ...
        -- ...
        -- ...
  -- sh
    -- k8s-cluster-prod-001
        -- ...
        -- ...
        -- ...
        -- ...
```

- 目录说明
    - argocd-manifests为argocd进行同步的目录。
    - bj/gd/hk/sh为各个区域。
    - k8s-cluster-prod-001，表示一个k8s集群名称，有可能还会有test集群，或者托管的集群。
    - k8s-cluster-prod-001目录下的就表示各个具体的应用了。
